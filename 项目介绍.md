# 项目概况

Brotato-like 幸存者类游戏（赛博土豆·Cyberblade），纯前端 Canvas 渲染，CSV 驱动全数据。

# 技术栈

- **纯前端**：无框架，无构建工具，Canvas 2D 渲染
- **运行方式**：`file://` 直接打开 `index.html`（离线可玩）
- **测试**：Vitest（Node.js 单元测试，不跑浏览器）
- **Node.js**：数据构建脚本（csv2json → data-bundle）

# 数据管道

```
csv/*.csv ──► scripts/csv2json.cjs ──► src/data/*.json
                                               │
                              scripts/generate-data-bundle.js
                                               │
                                               ▼
                                    src/data/data-bundle.js
```

- **CSV 是单一数据源**，JSON 和 bundle 都是生成的
- 修改 CSV 后必须运行两脚本：`node scripts/csv2json.cjs` → `node scripts/generate-data-bundle.js`
- 运行时 bundle 在 `window.__DATA_BUNDLE__` 中

# 核心系统

## 角色数据（`csv/characters.csv`）

每个角色有 32 列：id, name, desc, icon, unlocked, maxWeapons, 基础属性, tags, unlock条件, passives, startingWeapons, preferredClasses

关键字段：
- `tags`：角色标签（melee|gun|bow|magic|medic|lance|tech），决定武器选单显示哪些武器
- `startingWeapons`：初始武器 ID（管道符分隔），武器选单默认选中
- `passives`：被动技能 ID
- `engineering`：工程值，影响炮塔伤害

## 武器数据（`csv/weapons.csv`）

50+ 武器，46 列。关键设计：
- `tag`：关联 synergy 标签（melee|gun|bow|magic|medic|lance）
- `class`：武器大类（Blade|Blunt|Precise|Elemental|Heavy|Medical|Support|Primitive|Tool）
- `behavior`：攻击行为（melee_sweep|melee_thrust|bullet|spread|explode|shock|homing|spray|heal_bullet）
- 每种武器有 4 级数值（damage_lv1~4, cooldown_lv1~4）

## 协同天赋（`csv/synergies.csv`）

标签 + 阈值（2/4/6）→ 属性加成。7 种标签：
- melee, ranged, fire, explosive, crit, **tech**, economy
- `tech` 标签专为工程师设计：2=+5工程, 4=+10工程+1炮塔, 6=+20工程+2炮塔+30%炮塔伤

## 渲染（`src/engine/renderer.js`）

- Canvas 2D，CanvasRenderingContext2D
- 每帧重绘：背景 → 容器 → 拾取物 → 宝箱 → 敌人 → 粒子 → 玩家 → 子弹 → 炮塔
- 武器环绕角色旋转，使用 PNG 贴图或彩色圆点降级

## 波次系统（`src/engine/wave.js`）

- Budget 制 + 4 种刷怪模式（random/circle/fixed/wave）
- 20 关上限，可无尽模式
- 每波结束 → 宝箱 → 升级（2选1）→ 商店

# 重要机制

## 弹道系统（`weaponBulletTypes.csv`）

武器弹道视觉由 `behavior` 或 `tag` 匹配配置表：
- shape: bolt/arrow/beam/circle
- 支持 `image` 字段（`assets/bulletTypes/*.png`），优先于程序绘制
- 降级链：精确匹配 → 通配 `_default` → 程序默认

## 医疗箱（`container.js` + `healthPickup.js`）

每波生成 2-4 个容器，子弹/近战击破后掉落医疗包：
- 回血 15-25，8 秒消失，拾取范围吸引
- 近战回退目标：无敌人时自动攻击医疗箱

## 炮塔系统（`turret.js`）

- 每波开局统计 Tool 类武器数 + synergy turretCount → 生成炮塔
- 自动索敌 300 范围，0.73s 射击间隔
- 伤害公式：`(10 + 80% × engineering) × turretDamage`
- 炮塔子弹独立管理，橙黄色发光圆形

# 工程原则

- 数据驱动优先（CSV 唯一源）
- 武器不可超过 lv4
- 图片资源优先于程序绘制
- 测试全绿才能提交
