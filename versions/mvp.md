# CYBER BLADE · MVP 版本策划文档

> 版本：**MVP**（Minimum Viable Product）
> 状态：✅ 已开发完成

## 一、版本目标

**核心定位**：可玩的核心循环验证 — 证明玩法模型成立，数值体系合理，能为后续版本提供稳定的基础。

### MVP 验收标准

| 编号 | 标准 | 状态 |
|------|------|------|
| 1 | 角色选择 → 战斗 → 商店 → 升级 → 循环 完整跑通 | ✅ |
| 2 | 至少 3 个可选角色 | ✅ 赛博游侠/重型机甲/疾影刺客 |
| 3 | 10 种武器，8 种不同射击行为 | ✅ |
| 4 | 12 种可购买道具 | ✅ |
| 5 | 6 种敌人类型，含精英/BOSS | ✅ |
| 6 | 无限关卡 + 难度递增 | ✅ |
| 7 | 跨局解锁系统 | ✅ |
| 8 | 武器标签羁绊系统 | ✅ |
| 9 | 数值体系带上限/递减/百分比缩放 | ✅ |
| 10 | 图标资源完成 | ✅ ComfyUI 生成 31 个 |
| 11 | 零 JS 错误，帧率稳定 | ✅ |

### 6.2 难度缩放公式

```
scale = 1 + (currentLevel - 1) * 0.15
enemy.hp *= scale
enemy.damage *= scale
enemy.speed *= (1 + (scale - 1) * 0.3)
生成间隔 = max(0.4, 1.5 - (level - 1) * 0.04)
敌人数 = 4 + floor(level * 1.5)
```

### 7.2 护甲减伤公式（递减曲线）

```
damageReduction = armor / (armor + 50)

护甲  →  减伤率
  0   →   0%
 25   →  33%
 50   →  50%
 75   →  60%
100   →  66%
```

## 十、技术架构

### 10.1 技术栈
- **渲染**：HTML5 Canvas2D
- **UI**：DOM + CSS（backdrop-filter 模糊）
- **字体**：Google Fonts (Orbitron/Rajdhani)
- **持久化**：localStorage
- **零依赖**，除 Google Fonts 外无外部资源

### 10.2 文件结构（13 个源文件）

```
src/
├── input.js       # 输入系统
├── stats.js       # 数值体系 v2（上限/递减/等级选项）
├── unlock.js      # 跨局解锁
├── character.js   # 角色系统
├── bullet.js      # 子弹 + 对象池
├── particle.js    # 粒子特效 + 对象池
├── player.js      # 玩家系统
├── enemy.js       # 敌人 + AI
├── wave.js        # 关卡生成
├── shop.js        # 商店 + 羁绊
├── ui.js          # UI 系统
├── assets.js      # 图标加载
├── renderer.js    # 渲染管线
└── main.js        # 游戏引擎
```

### 10.3 性能策略
- 对象池（子弹、粒子）
- dt 限幅 0.05s
- 局部网格渲染
- CSS backdrop-filter 替代 Canvas 模糊
