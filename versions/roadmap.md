## 三、Beta1 — 内容扩充 + 体验打磨

> **目标**：丰富游戏内容深度，让玩家有更多构筑可能性和敌人的挑战性。

### 3.1 武器扩充（+5 种 → 15 种）

| 新武器 | 行为 | 标签 | 特点 |
|--------|------|------|------|
| 回旋镖 | boomerang | melee | 飞出后返回，可穿透 |
| 重力枪 | gravity | tech | 将敌人拉向命中点 |
| 纳米虫群 | swarm | tech | 释放自动攻击附近的虫群 |
| 电磁脉冲 | emp | heavy | 大范围爆炸 + 短暂眩晕 |
| 火焰喷射器 | flamethrower | rapid | 持续扇形火焰，叠加灼烧 |

**需求：新增 3 种武器行为（boomerang/gravity/swarm），5 个新图标**

### 3.2 道具扩充（+6 种 → 18 种）

| 新道具 | 效果 | 定位 |
|--------|------|------|
| 荆棘甲 | 受到伤害时反弹 50% 给攻击者 | 反伤流 |
| 能量盾 | 每 10 秒生成一次护盾，抵挡 1 次伤害 | 防御流 |
| 弹药背包 | 子弹 +2（但散布 +20%） | 代价型 |
| 兴奋剂 | 攻速 +25%，移速 +20%，受伤 +20% | 高风险高回报 |
| 磁暴线圈 | 靠近的敌人每 2 秒受 10 点伤害 | 近身的代价 |
| 复制器 | 20% 概率射出双倍子弹 | 概率型 |

### 3.3 敌人扩充（+3 种 → 9 种）

| 新敌人 | 行为 | 特点 |
|--------|------|------|
| 分裂者 | chase | 死亡后分裂为 2 个小个体，HP 减半 |
| 护盾兵 | chase | 正面有护盾，需绕后攻击 |
| 召唤者 | summon | 每隔几秒召唤 2 个 basic 敌人 |

### 3.4 Boss 机制重做

- 每个 Boss 有**独特技能**，不仅是数值放大
- 示例：BOSS·毁灭者 → 技能：地面震荡波（圆形范围）+ 召唤小兵
- 每 5 关 Boss 的 HP/伤害递增曲线更陡

### 3.5 武器平衡性调整

- 根据 MVP 测试数据，调整武器价格/修正/解锁条件
- 确保低槽位武器（1槽）更具竞争力
- 确保高槽位武器（3槽）对得起占用

### 3.6 关卡机制丰富

| 新机制 | 说明 |
|--------|------|
| 波次标签 | 每关随机一个标签（"敌人加速"/"金币翻倍"/"精英更多"） |
| 奖励波 | 每 3 关出现一波奖励波，敌人弱但金币多 |
| 危险波 | 每 7 关出现一波危险波，敌人增强但掉落更好 |

### 3.7 UI 优化

- 商店中显示武器 DPS 预估值
- 属性面板增加各级属性叠加来源说明
- 结算界面增加时间统计、DPS 统计

## 四、Beta2 — 音效 + 视觉升级

> **目标**：从"可玩"提升到"有沉浸感"。

### 4.1 音效系统

| 类型 | 数量 | 说明 |
|------|------|------|
| 武器音效 | 10+ | 每种武器独特射击声 |
| 击中音效 | 5+ | 命中/暴击/击杀/护盾/未命中 |
| 环境音效 | 3+ | 背景白噪音、关卡开始/结束 |
| UI 音效 | 5+ | 点击/购买/升级/解锁/结算 |

**技术方案**：Web Audio API + 程序化生成（初期可免版权音效库）

### 4.2 视觉特效升级

| 改进 | 说明 |
|------|------|
| 武器弹道特效 | 每种武器独特弹道颜色/轨迹/尾迹 |
| 命中特效 | 暴击火花 / 护甲弹开 / 冰霜碎裂 |
| 击杀特效 | 不同敌人类型不同死亡效果 |
| 屏幕震动 | 爆炸/重武器开火时轻微屏幕震动 |
| 低血量特效 | 屏幕边缘红色脉动 + 心跳效果 |
| 升级特效 | 全屏闪光 + 光柱升级效果 |
| BOSS 登场 | 屏幕震动 + 暗色转场 + 名称显示 |

### 4.3 HUD 升级

| 改进 | 说明 |
|------|------|
| 小地图 | 右上角缩略地图，显示敌人/材料/边界 |
| 武器图标栏 | HUD 底部显示当前所有武器图标 |
| 伤害数字 | 命中时弹出浮动伤害数字（可开关） |
| 经验获得提示 | XP 获得时浮动提示 |

### 4.4 动画系统

- 角色 idle/受伤闪烁动画
- 敌人行走动画（简单帧动画）
- 商店卡片入场动画（从两侧滑入）
- 升级选项渐变出现

## 五、Beta3 — 平台适配 + 性能

> **目标**：覆盖更多设备和场景。

### 5.1 触屏支持

- 虚拟摇杆（左半边屏幕）
- 自动攻击 / 手动选择目标
- UI 元素触控尺寸适配

### 5.2 响应式优化

| 设备 | 适配内容 |
|------|---------|
| 手机竖屏 | 单手操作 / 简化的 HUD |
| 手机横屏 | 全屏游戏模式 |
| 平板 | 优化触控布局 |
| 桌面大屏 (4K) | UI 缩放比例优化 |

### 5.3 性能优化

- 粒子数量动态调整（根据设备性能）
- Canvas 离屏渲染优化
- 对象池扩容策略优化
- 首次加载时的图片预加载进度条

### 5.4 国际化

- 英文版本（i18n 框架）
- 文本与数据分离

## 六、v1.0 正式版 — 发布准备

> **目标**：可以公开发布的可玩版本。

### 6.1 发布渠道

- **Web 版**：部署到 GitHub Pages / Vercel / Netlify
- **Itch.io**：作为独立游戏发布
- **手机 Web 版**：PWA 安装

### 6.2 发布前检查清单

| 项目 | 说明 |
|------|------|
| 教程/引导 | 首次游玩时的新手引导流程 |
| 游戏说明页 | 操作/系统/数值说明 |
| 加载页面 | 资源加载时的进度条和趣味提示 |
| 错误处理 | 全局错误捕获 + 友好提示 |
| 分析 | 简单的事件追踪（可选） |
| 隐私 | 无数据收集声明 |
| SEO | 页面 meta 优化 |
| 社交分享 | Open Graph 卡片预览 |

### 6.3 发布后规划

- 收集玩家反馈
- 社区 Discord / GitHub Issues
- 持续平衡性更新
- 可考虑扩展包：新角色/武器/模式
