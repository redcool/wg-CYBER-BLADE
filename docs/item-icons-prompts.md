## 使用建议

- 所有 prompt 统一加前缀：`game item icon, roguelike inventory icon,`
- 如需透明底：追加 `transparent background,`
- 如需小图标：追加 `32x32 pixel perfect,`
- 如需强化像素感：追加 `simple, flat shading, retro palette`
