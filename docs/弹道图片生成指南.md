# 弹道图片生成指南

> 存放路径: `assets/bulletTypes/*.png`

游戏已内置程序绘制回退（无图也不崩），但贴图能显著提升视觉品质。

## 图片规格

| 属性 | 要求 |
|------|------|
| 格式 | PNG 透明背景 |
| 尺寸 | 32×32 像素（每个弹道居中） |
| 命名 | 见下方列表 |
| 风格 | 像素风或卡通扁平，与武器图标协调 |

## 全部弹道图片清单

### 通用核心（优先做这 3 张可覆盖大部分）

| 文件名 | 位置 | 颜色方案 | 描述 |
|--------|------|---------|------|
| `bullet_default.png` | 通用子弹兜底 | 黄白 (#ffff44) | 发光小圆球，白色核心+黄色光晕，4px大小 |
| `spread_default.png` | 散射兜底 | 黄白 (#ffff44) | 同上但稍小（3px），可做微椭圆 |
| `explode.png` | 爆炸弹 | 红橙 (#ff4444/#ff6600) | 火焰/火球形态，红橙渐变，8px 略大 |

### 子弹行为 - 按武器标签

| 文件名 | 行为 | 颜色 | 描述 |
|--------|------|------|------|
| `bullet_gun.png` | 枪械子弹 | 白 (#ffffff) | 流线型弹头/小圆点，带白色尾迹 |
| `bullet_bow.png` | 弓箭 | 白 (#ffffff) | 箭矢形状，菱形箭头+细杆 |
| `bullet_magic.png` | 魔法弹 | 淡黄 (#ffffaa) | 能量球体，半透明发光，比圆形更多芒刺感 |
| `bullet_medic.png` | 医疗弹 | 淡绿 (#aaffaa) | 绿色十字或心形发光球 |
| `bullet_lance.png` | 长枪 | 金黄 (#ffdd44) | 矛尖形状，菱形细长 |
| `bullet_melee.png` | 近战 | 金黄 (#ffdd44) | 弧光/斩击特效，半月形 |

### 散射行为

| 文件名 | 颜色 | 描述 |
|--------|------|------|
| `spread_gun.png` | 白 (#ffffff) | 小圆弹头，比 bullet_gun 略小 |
| `spread_bow.png` | 白 (#ffffff) | 短箭矢，比 bullet_bow 短小 |
| `spread_magic.png` | 淡黄 (#ffffaa) | 小能量碎片，不规则边缘 |

### 冰冻行为

| 文件名 | 颜色 | 描述 |
|--------|------|------|
| `frost_bow.png` | 蓝白 (#88ddff) | 冰晶箭矢，半透明冰蓝色 |
| `frost_magic.png` | 蓝青 (#88ddff/#aaffff) | 冰晶/冰棱形状，光束状 |
| `frost_default.png` | 蓝白 (#88ddff) | 冰晶碎片，六角形或菱形 |

### 追踪行为

| 文件名 | 颜色 | 描述 |
|--------|------|------|
| `homing_bow.png` | 亮黄 (#ffff44) | 发光箭矢，尾迹带拖尾 |
| `homing_magic.png` | 黄白 (#ffff44/#ffffaa) | 追踪能量球，带小尾巴或粒子尾迹 |
| `homing_default.png` | 亮黄 (#ffff44) | 发光小圆球，带方向指示 |

### 特殊行为

| 文件名 | 颜色 | 描述 |
|--------|------|------|
| `shock.png` | 橙黄 (#ffaa44) | 闪电/电击标记，锯齿形，不一定要做贴图（已有程序绘制闪电） |
| `spray.png` | 白 (#ffffff) | 锥形粒子，细小圆点集群（程序已绘制锥形，贴图可选） |
| `heal_bullet.png` | 亮绿 (#00ff88) | 绿色心形或十字发光球 |

## 建议优先级

1. **3 张通用核心**：`bullet_default`, `spread_default`, `explode` → 覆盖 60%+ 弹道
2. **6 张基础子弹**：`bullet_gun`, `bullet_bow`, `bullet_magic`, `bullet_medic`, `bullet_lance`, `bullet_melee`
3. **其余按需**：散射/冰冻/追踪/特殊

## 技术说明

- 渲染逻辑在 `src/engine/renderer.js` 的 `_getBulletConfig` 和 `drawBullet` 方法
- 加载时从 `assets/bulletTypes/{image}.png` 读取，失败时走程序绘制（shape/color/size 参数）
- 图片文件名与 `csv/weaponBulletTypes.csv` 的 `image` 列对应
