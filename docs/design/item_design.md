# 道具系统策划文档

> **参考来源：** 《土豆兄弟》(Brotato) 道具系统
> **设计目标：** 每个道具都有明确的定位与数值，部分道具附带负面效果增加策略深度

---

## 一、设计原则

1. **标签机制**：每个道具属于特定类别，道具可多次购买叠加效果（非unique）
2. **Unique标记**：独特道具具有特殊机制，只能购买一次，购买后不出现在商店
3. **数值叠加**：非unique道具可重复购买，效果线性叠加（注意属性上限）
4. **负面效果**：部分强力道具附带负面效果，需要玩家策略取舍
5. **价格梯度**：基础道具 4-10 金币，强力/独特道具 8-16 金币

## 四、特殊机制说明

1. **能量屏障**：受到伤害前判定，成功格挡后开始8秒冷却
2. **荆棘装甲**：受到伤害时，反弹伤害的30%给攻击者（仅近战攻击者）
3. **反应装甲**：受伤后有50%概率触发，2秒内回复5HP，冷却3秒
4. **存钱罐**：关闭商店进入战斗时，按当前金币的15%获得利息（向上取整，最低1）
5. **折扣券**：商店中所有物品价格-2（不影响刷新费用），价格最低为1
6. **燃烧扩散器**：身上有燃烧层数的敌人死亡时，将燃烧传播给半径200内的敌人
7. **狂战士之血**：HP低于30%时自动激活攻速+50%、伤害+30%
