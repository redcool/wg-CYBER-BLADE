## 一、武器数据

### 通用公式

- **最终伤害** = `player.damage * damageMult * (暴击? critMultiplier : 1)`
- **攻击冷却** = `(1.0 / player.attackSpeed) * attackSpeedMult`
- **子弹速度** = `bulletSpeed`（固定值，不受玩家属性影响）
- **穿透次数** = `pierce`（0 = 命中即消失）

### 难度缩放
```
scale = 1 + (currentLevel - 1) * 0.15
enemy.hp *= scale
enemy.damage *= scale
enemy.speed *= (1 + (scale - 1) * 0.3)
```

### 生成速率
```
baseInterval = 1.5s
reduction = min(0.8, (level - 1) * 0.04)
finalInterval = max(0.4, baseInterval - reduction)
```

## 五、武器羁绊系统

### 武器标签

每把武器归属一个标签（tag），装备同标签武器达到一定数量触发羁绊加成。

| 标签ID | 名称 | 图标 | 涉及武器 |
|--------|------|------|---------|
| rapid | 速射 | ⚡ | 基础手枪、加特林、跟踪导弹 |
| heavy | 重火 | 💥 | 散弹枪、狙击枪、火箭筒 |
| tech | 科技 | 🔮 | 激光炮、电击枪、冰霜枪 |
| melee | 近战 | ⚔️ | 等离子刀 |

### 羁绊效果

| 标签 | 门槛 | 加成 |
|------|------|------|
| ⚡ 速射 | 2把 | 攻速 +15% |
| ⚡ 速射 | 3把 | 攻速 +30%、弹速 +20% |
| 💥 重火 | 2把 | 伤害 +20% |
| 💥 重火 | 3把 | 伤害 +40%、穿透 +1 |
| 🔮 科技 | 2把 | 暴击率 +8% |
| 🔮 科技 | 3把 | 暴击率 +15%、伤害 +15% |
| ⚔️ 近战 | 1把 | 吸血 +5%、暴击伤害 +50% |

> 注意：羁绊加成是乘法叠加到玩家当前属性的。例如 2x 重火激活后，最终伤害 = 当前伤害 × 1.2。
