# 道具图片生成指南

> 存放路径: `assets/items/*.png`

游戏内道具 PNG 缺失时仅显示 `I` 文本兜底，不会报错。制作后放入 `assets/items/` 即可自动识别。

## 图片规格

| 属性 | 要求 |
|------|------|
| 格式 | PNG 透明背景（ComfyUI 黑色背景可后处理去除） |
| 尺寸 | 64×64 像素，主体居中 |
| 命名 | 见下方列表，文件名 = CSV 中的 `id` 字段 |
| 风格 | 扁平卡通/游戏道具图标风格，与现有 19 张道具协调 |

## 批量生成建议

ComfyUI 批量工作流设置建议：

1. **Checkpoint**: 卡通/扁平风格模型（如 animagine XL 或类似）
2. **分辨率**: 64×64
3. **统一后缀提示词**: `game item icon, simple flat style, centered on dark background, precise lines`
4. **批量模式**: 用 CSV 中的 `id` 作为文件名输出

生成后直接放入 `assets/items/`，游戏会自动加载，无需修改代码。
