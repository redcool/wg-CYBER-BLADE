# 《土豆兄弟》(Brotato) 竞品分析报告

> 分析时间：2026年5月 | 参考来源：Brotato Wiki, Steam社区, Reddit

---

## 一、游戏概况

**Brotato**（土豆兄弟）是一款"类吸血鬼幸存者" + **自走棋羁绊** 结合的自动射击生存 Roguelite。

核心公式：
```
自动射击 + 商店构筑 + 武器合成 + 羁绊加成 + 随机事件
```

---

## 二、核心系统深度解析

### 2.1 属性系统

Brotato 的属性比我们更细分：

| 属性 | 说明 | 我们当前 |
|------|------|---------|
| **Max HP** | 最大生命 | ✅ |
| **HP Regen** | 每秒回复固定数值 | ✅ |
| **Life Steal** | 攻击吸血百分比 | ✅ |
| **Melee Damage** | 近战伤害加成 | ❌（未区分近战/远程） |
| **Ranged Damage** | 远程伤害加成 | ❌ |
| **Elemental Damage** | 元素伤害加成 | ❌ |
| **Attack Speed** | 攻击速度 | ✅ |
| **Crit Chance** | 暴击率 | ✅ |
| **Crit Damage** | 暴击伤害倍率 | ✅ |
| **Armor** | 护甲（百分比减伤） | ✅ |
| **Dodge** | 闪避率 | ✅ |
| **Speed** | 移动速度 | ✅ |
| **Range** | 攻击范围 | ✅ |
| **Engineering** | 工程学（炮台/地雷伤害） | ❌ |
| **Harvesting** | 每关结束额外材料 | ✅（类似） |
| **Luck** | 幸运（商店品质/掉落率） | ✅ |
| **Pickup Range** | 拾取范围 | ✅ |
| **Armor Penetration** | 护甲穿透 | ❌ |

> 💡 **差距**：我们缺少近战/远程/元素伤害的区分，以及工程学体系。

### 2.2 武器系统

Brotato 最核心的设计差异：

**① 武器合成系统**
```
两把同名称同等级的武器 → 合成更高级一把
Tier 1 (白) → Tier 2 (蓝) → Tier 3 (紫) → Tier 4 (红)
```
- 武器等级提升不仅加伤害，还可能解锁新效果
- 鼓励玩家"多买同种武器"来合成升级

**② 职业标签 (Tags) / 羁绊系统**

武器有标签如：`Primitive`, `Blade`, `Gun`, `Tool`, `Elemental`, `Heavy`, `Melee`, `Precision` 等。
装备 **3/6 件同标签** 武器触发羁绊加成：

| 标签 | 3件加成 | 6件加成 |
|------|---------|---------|
| Blade | +20% 近战伤害 | +50% 近战伤害 |
| Gun | +20% 远程伤害 | +50% 远程伤害 |
| Elemental | +20% 元素伤害 | +50% 元素伤害 |
| Heavy | -10% 攻速 / +30% 伤害 | ... |
| Precision | +10% 暴击率 | ... |

**③ 属性缩放 (Scaling)**
- 每把武器有各自的属性缩放系数
- 例如：AK47 有 65% 远程伤害缩放 + 35% 元素伤害缩放
- 这意味着你堆的属性会以不同比例作用于不同武器

> 💡 **差距**：我们只有"攻击力"一个维度，没有武器合成和羁绊系统。

### 2.3 道具系统

Brotato 道具设计哲学：**Buff 必配 Debuff**

```
+30% 伤害    但    -10% 生命
+15% 攻速    但    -5% 护甲
+200% 收获   但    -50% 伤害
```

每个道具都是一个"风险 vs 收益"的选择题，构筑深度由此产生。

> 💡 **差距**：我们的道具都是纯增益，缺少"有代价的强力道具"。

### 2.4 角色系统

Brotato 有 **60+** 个角色，每个角色有极端差异化的被动：

| 角色 | 被动 | 武器槽 |
|------|------|--------|
| 全能者 (Generalist) | +10%所有属性 | 6 |
| 单臂 (One-Armed) | +100%伤害 | 1 |
| 多面手 (Multitasker) | 可装备12把武器，但所有属性-20% | 12 |
| 工程师 (Engineer) | 工程学+200%，但攻速-50% | 6 |
| 公牛 (Bull) | 受到伤害时爆炸，HP极高 | 6 |
| 大嗓门 (Loud) | 更多敌人 = 更多材料掉落 | 6 |

> 💡 **差距**：我们只有3个角色，且差异化不够极端。

### 2.5 经济系统

Brotato 的关键设计：
- **材料 = 经验值** —— 捡材料既加钱又加经验，玩家必须权衡"冒险去捡 vs 保命"
- **收获 (Harvesting)** 属性每波结束额外给材料，是滚雪球的关键
- **银行利息**：保留材料不花，每波获得利息收入

> 💡 **借鉴点**：我们可以让金币同时提供少量经验值。

### 2.6 商店系统

```
每波结束 → 商店:
- 刷新 6 个商品（武器 + 道具混合）
- 可花钱重新刷新
- 只买对的，不买贵的（资源有限）
- 锁定商品留到下一波
``` 

> 💡 **借鉴点**：锁定商品功能、更丰富的刷新策略。

### 2.7 难度系统

Brotato 的难度梯度非常清晰：

| 难度 | 名称 | 敌人倍率 | 解锁方式 |
|------|------|---------|---------|
| D0 | 安全 | 1.0x | 默认 |
| D1 | 中等 | 1.2x | D0通关 |
| D2 | 困难 | 1.5x | D1通关 |
| D3 | 极难 | 1.8x | D2通关 |
| D4 | 疯狂 | 2.2x | D3通关 |
| D5 | 绝望 | 2.8x | D4通关 |

> 💡 **差距**：我们没有难度选择系统。

### 2.8 随机事件

Brotato 在商店之后、战斗之前，有时会出现**随机事件**：
- 选择 "获得一个随机武器" 或 "获得200材料"
- 选择 "损失10%HP换取200%伤害持续10秒"
- 选择 "下波敌人翻倍，但材料掉落3倍"

> 💡 **借鉴点**：可以加随机事件增加每局变化。

## 三、与《赛博土豆》的差距对比

| 维度 | Brotato | 赛博土豆（当前） | 优先级 |
|------|---------|----------------|--------|
| 角色数量 | 60+ | 3 | 🔴 低（不需要那么多） |
| 角色差异化 | 极端，改变玩法 | 温和的数值差异 | 🔴 需要加强 |
| 武器合成 | ✅ 4级合成 | ❌ 无 | 🟡 可以考虑 |
| 武器羁绊 | ✅ 标签系统 | ❌ 无 | 🔴 最有价值的改进 |
| 属性维度 | 15+ 细分 | 10+ | 🟡 可以增加 |
| 有代价的道具 | ✅ Buff+Debuff | ❌ 纯增益 | 🟡 可以增加 |
| 难度选择 | D0-D5 | ❌ 无 | 🟡 可以考虑 |
| 随机事件 | ✅ 有 | ❌ 无 | 🟢 有趣但不急 |
| 解锁深度 | 成就驱动深度高 | 条件解锁较浅 | 🟡 可以加强 |
| 战斗手感 | 平滑 | 已验证OK | ✅ |
| 30秒关卡 | ❌ (20波固定) | ✅ 30秒无限 | 差异化优势 |
| 多种角色机制 | 丰富被动 | 基础数值差异 | 🟡 可以改进 |

## 四、改进建议（按 ROI 排序）

### 🔥 高优先级

**1. 武器标签 + 羁绊系统**
- 给每把武器打标签：`实弹` / `能量` / `近战` / `重武` / `冰霜` 等
- 装备 2/4 把同标签武器获得额外加成
- 激励玩家走特定流派，而非"有啥买啥"

**2. 武器稀有度/合成**
- 武器分白→蓝→紫→金 4 级
- 购买两把同名武器自动合成升级
- 稀有度影响伤害倍率和额外效果

**3. 有代价道具**
- 增加"强效但不稳定"的道具
- 例如：+50%伤害但-20%闪避、+200%收获但-50%回复
- 让每个购买决定都有权衡

### 🟡 中优先级

**4. 属性细分**
- 将 attackDamage 拆分为 `近战伤害` / `远程伤害`
- 武器根据类型应用不同属性缩放系数
- 增加"护甲穿透"属性

**5. 角色被动机制强化**
- 给每个角色增加一个独特被动
- 例如：游侠→每5秒闪避下一次攻击、机甲→每10秒范围震击

**6. 难度系统**
- 通关后解锁更高难度
- 不同难度影响敌人强度 + 奖励倍率

### 🟢 低优先级

**7. 随机事件**
- 商店后随机出现事件选择
- 增加每局变化

**8. 银行/利息机制**
- 未花完的金币每波产生 5% 利息
- 鼓励玩家策略性存钱

## 五、总结

《土豆兄弟》成功的关键因素：

1. **构筑深度**—— 羁绊+合成+有代价道具 = 每局都有不同的Build
2. **极端角色** —— 每个角色是一种全新玩法
3. **清晰难度阶梯** —— 给人以"变强"的正反馈
4. **成就驱动的解锁** —— 每次通关都有目标感

**我们当前的差异化优势：**
- 无限关卡（而非固定20波）
- 30秒快速节奏
- 多种武器行为（近战/爆炸/连锁/冰霜/跟踪等）—— Brotato 的武器行为比较单一
- 赛博朋克美术风格

> 核心改进方向：**在保留多武器行为和快速节奏优势的基础上，引入羁绊系统 + 有代价道具 + 武器合成**，大幅提升构筑深度。
