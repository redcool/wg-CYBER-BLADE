# Brotato (土豆兄弟) 数值设计参考文档

> 本文档收集 Brotato 原版游戏的核心数值系统，用于指导本游戏的武器/道具平衡设计。

---

## 1. 武器系统

### 武器分类与标签

| 标签 | 名称 | 核心机制 |
|------|------|----------|
| Blade | 刀剑 | 近战，暴击 |
| Blunt | 钝器 | 近战，击退 |
| Primitive | 原始 | 近战，数量加成 |
| Unarmed | 徒手 | 极速攻击 |
| Gun | 枪械 | 远程，精准 |
| Precise | 精准 | 远程，暴击穿透 |
| Explosive | 爆炸 | AOE 范围 |
| Elemental | 元素 | 元素伤害/特效 |
| Tool | 工具 | 炮台/工程 |
| Medical | 医疗 | 治疗/辅助 |
| Ghost | 幽灵 | 特殊成长 |
| Ethereal | 以太 | 魔法伤害 |

### 武器核心属性

| 属性 | 说明 | Brotato 典型范围 |
|------|------|------------------|
| 基础伤害 | 武器基础伤害 | 8~45 |
| 攻速 (Attack Speed) | 攻击间隔，以秒计 | 0.15~1.7s |
| 暴击率 (Crit Chance) | 默认 3% | 3%~15% |
| 暴击倍率 (Crit Multiplier) | 默认 ×2 | ×2~×3 |
| 射程 (Range) | 近战 100~350, 远程 600~1200 | 100~1200 |
| 穿透 (Pierce) | 子弹穿透次数 | 0~5 |

### 武器等级与品质

Brotato 中武器可通过购买同种武器升级：
- 等级每+1 → 伤害 ×1.25
- 品质: T1(普通) → T2(优秀) → T3(稀有) → T4(传说)
- 品质影响伤害倍率和售价

### 武器羁绊系统

每个武器有标签，装备同标签武器达到阈值获得额外加成：

**近战 Blade:** 2把 +5%吸血/+50%暴伤 → 3把 +10%吸血/+100%暴伤 → 4把 +15%吸血/+150%暴伤/+3护甲

**枪械 Gun:** 2把 +15%射程 → 3把 +30%射程/+15%弹速 → 4把 +45%射程/+30%弹速/+1子弹

**弓箭 Bow:** 2把 +10%暴率 → 3把 +15%暴率/+20%暴伤 → 4把 +20%暴率/+40%暴伤/+1穿透

**元素 Elemental:** 2把 +15%伤害 → 3把 +25%伤害/+20%减速 → 4把 +35%伤害/+30%减速/+1连锁

**医疗 Medical:** 2把 +1HP/秒回复 → 3把 +2HP/秒回复/+5%吸血 → 4把 +3HP/秒回复/+10%吸血/+20HP

## 3. 商店系统

### 刷新机制
- 每次战斗后进入商店
- 基础刷新花费：首次 1 金币，每次+1（1→2→3...）
- 购买全部4件商品后免费刷新一次
- 幸运属性影响高品质出现概率

### 商品池
- 商店同时出现武器和道具（混合池）
- 武器基于角色适配标签过滤
- 独特道具 (unique) 已购买后不再出现
- 锁定物品跨商店保留

- 医疗兵: 回复+2/秒，治疗效果+25%

## 6. 减伤公式

### 护甲减伤（递减曲线）
```
有效伤害 = 原始伤害 / (1 + 护甲 × 0.1)
```
即 10 护甲 → 50% 减伤，20 护甲 → 66% 减伤

### 闪避判定
```
if random() < 闪避率 → 完全闪避
```
闪避上限 60%

## 7. 平衡参考

### 武器价值参考
- 1槽位武器：基础 15 伤害/秒
- 2槽位武器：基础 25 伤害/秒  
- 3槽位武器：基础 35 伤害/秒

### 伤害计算公式
```
最终伤害 = 基础伤害 × 伤害倍率 × (1 + 角色伤害加成) × (1 + 羁绊伤害加成)
暴击时: 最终伤害 × 暴击倍率
```

### 攻速与冷却
```
攻击间隔 = 1.0 / 攻速 × 武器攻速倍率
```
