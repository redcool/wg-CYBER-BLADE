# 土豆兄弟\(Brotato\)数值设计文档

## 一、数值设计概述

《土豆兄弟》的数值体系采用 \\*\\* 线性成长 \+ controlled inflation（受控通货膨胀）\\*\\* 的核心设计，通过清晰的公式化数值，保证了局内养成的可预期性，同时通过掉落衰减、价格通胀等机制，避免后期数值膨胀失控，同时通过保底机制平衡随机养成的挫败感。

整体数值设计的核心目标：

1. 前 5 波为构筑期，通过保底机制保证玩家快速成型 Build

2. 中期通过线性成长保证战力与怪物强度同步提升

3. 后期通过通胀、掉落衰减控制经济与战力的增长速度，避免数值爆炸

4. 高风险高回报的数值设计（如诅咒、收获流）提供极限玩法的空间

#### 升级所需经验公式

```Plain Text
升级所需经验 = (当前等级 + 3)²
```

例如：

- 1 级升 2 级：\(1\+3\)²=16 点经验

- 5 级升 6 级：\(5\+3\)²=64 点经验

- 10 级升 11 级：\(10\+3\)²=169 点经验

## 四、经济与掉落数值

### 1\. 材料掉落衰减机制

为了控制后期经济膨胀，从第 5 波开始，怪物的材料掉落率逐步衰减，最低到 50%：

|波次|1\-4|5|10|15|20|25|30\+|
|---|---|---|---|---|---|---|---|
|掉落率|100%|92\.5%|85%|77\.5%|70%|59\.5%|50%|
|*规则：从第 5 波开始，每波衰减 1\.5%，直到 33 波后稳定在 50% 最低值*||||||||

### 2\. 收获属性数值

收获属性是波次结束后额外获得的材料 / 经验，其成长机制为：

```Plain Text
每波结束后，收获属性会额外获得：收获值 × (1 + 5% × 已激活次数)
```

即每波激活后，收获的收益会提升 5%，形成指数增长，这也是收获流的核心数值基础。

*注意：20 波之后收获的收益会逐步衰减，避免无尽模式下的数值爆炸*

### 3\. 特殊经济数值

- 存钱罐：每波开始时，获得当前材料的 20% 额外金币，20 波后失效

- 节俭者：存款可以转化为伤害，每 25 材料提升 1% 伤害，后期可以堆出高额伤害

## 五、商店数值

### 1\. 道具价格通货膨胀

商店的道具价格随波次逐步提升，实现受控的通货膨胀，公式为：

```Plain Text
最终价格 = (基础价格 + 当前波次 + 基础价格 × 0.1 × 当前波次) × 商店价格系数
```

例如：

- 基础价格 20 的 SMG，第 1 波价格：20\+1\+20×0\.1×1=23

- 基础价格 45 的道具，第 5 波价格：45\+5\+45×0\.1×5=72\.5，取整 72

### 2\. 刷新价格数值

商店的刷新价格随波次和刷新次数逐步提升，公式为：

```Plain Text
刷新增量 = 向下取整(0.4 × 当前波次)，最低1
首次刷新价格 = 向下取整(0.75 × 当前波次) + 刷新增量
后续每次刷新，价格额外增加刷新增量
```

前 20 波的刷新价格表：

|波次|1|2|3|4|5|10|15|20|
|---|---|---|---|---|---|---|---|---|
|首次刷新价格|1|2|3|4|5|11|17|23|
|每次刷新增量|1|1|1|2|2|5|7|9|

### 3\. 商店稀有度概率

商店的道具稀有度随波次成长，同时受幸运属性影响：

|稀有度|最早出现波次|基础每波概率提升|最大概率|
|---|---|---|---|
|Tier1（普通）|1|0%|100%|
|Tier2（优秀）|2|6%|60%|
|Tier3（稀有）|4|2%|25%|
|Tier4（史诗）|8|0\.23%|8%|

#### 幸运对稀有度的影响

幸运属性会按比例提升稀有度的概率：

```Plain Text
实际概率 = ((每波概率 × (当前波次 - 最小波次)) + 基础概率) × (100% + 幸运)
```

该概率不会超过该稀有度的最大概率。

例如：10 点幸运，会让所有稀有度的概率提升 10%，但不会超过最大上限。

## 六、属性与战斗数值

### 1\. 核心属性公式

#### 护甲减伤公式

```Plain Text
减伤率 = 护甲 / (护甲 + 15)
每点护甲提升6.66%的有效生命值
```

例如：15 点护甲，减伤率为 50%，有效生命翻倍。

#### 生命再生公式

```Plain Text
每秒生命回复 = 0.09 × 生命再生 + 0.1
```

生命再生的回复速度有上限，最高每秒 10 点生命。

#### 伤害计算公式

```Plain Text
最终伤害 = (武器基础伤害 + 对应类型点伤加成) × 全局伤害% × 攻击速度% × (1 + 暴击率 × (暴击倍率 - 1))
```

该公式保证了点伤、全局伤害、攻速、暴击四个属性的收益等价，引导玩家均匀分配属性。

### 2\. 幸运属性概率

幸运属性的具体影响：

1. 提升商店道具稀有度概率（见上文）

2. 提升怪物掉落道具、血包的概率

3. 提升升级选项的稀有度概率

## 七、难度与无尽模式数值

### 1\. 危险等级数值乘数

不同危险等级下，怪物的属性有不同的乘数：

|危险等级|0|1|2|3|4|5|
|---|---|---|---|---|---|---|
|怪物属性乘数|100%|110%|120%|130%|140%|150%|

同时，高危险等级会解锁精英怪、多 BOSS 等机制：

- 危险 2\+：解锁精英怪波次

- 危险 5\+：第 20 波出现 2 个 BOSS

### 2\. 无尽模式数值成长

无尽模式下，怪物的属性会额外增加无尽因子的成长：

```Plain Text
无尽因子 = 0.02 × (当前波次 - 20)
```

怪物的速度、血量、伤害都会乘以该因子，同时：

- 商店的刷新价格、道具价格都会乘以无尽因子，实现通胀的进一步提升

- 诅咒系统的概率也会随无尽因子提升，很快达到 100% 的诅咒怪物概率

## 八、DLC 诅咒系统数值

### 1\. 核心概率公式

```Plain Text
诅咒因子 = 诅咒 / (诅咒 + 50)
怪物被诅咒概率 = 50% × 诅咒因子
物品被诅咒概率 = 15% × 诅咒因子
```

### 2\. 诅咒怪物数值

被诅咒的怪物属性增幅：

```Plain Text
血量增幅 = 150% + 2% × 诅咒值，最高750%
伤害增幅 = +25%
速度增幅 = +15%
```

同时，被诅咒的怪物死亡后，会掉落 33% 额外的材料。

### 3\. 诅咒物品数值

被诅咒的物品 / 武器的强化幅度，由诅咒修正系数决定：

```Plain Text
修正系数 = 0.4 + 0.02 × min(当前波次,20) + 随机波动(-0.3,0.3)
```

- 增益效果：乘以修正系数，最高 2\.1 倍

- 减益效果：除以修正系数，最低 0\.47 倍

- 购买被诅咒物品后，会获得额外的诅咒值：`诅咒值 = 0\.04 × 物品基础价格 × 修正系数`

## 九、数值设计总结

《土豆兄弟》的数值体系通过清晰的线性成长、受控通胀、保底机制，实现了轻量化肉鸽的完美平衡：

1. **线性成长保证可预期**：怪物、道具、升级的数值都采用线性成长，玩家可以清晰预判战力提升

2. **通胀机制控制膨胀**：通过掉落衰减、价格通胀，避免后期经济与战力的爆炸式增长

3. **保底机制降低挫败**：前 5 波的构筑期保底，保证玩家不会因为纯随机直接暴毙，降低上手门槛

4. **极限玩法的空间**：通过收获、诅咒等特殊数值机制，给高玩提供了极限挑战的空间，延长了游戏的重复游玩寿命
